# World Cup 2026 — Live Bracket Seeding

A single-page tracker for the 2026 FIFA World Cup: live group standings and an
auto-seeded knockout bracket you can fill in yourself.

**Live site:** _add your GitHub Pages URL here_

## What it does
- **Live standings** — pulled from ESPN's free feed and refreshed every minute, including in-progress games (with scorers).
- **Auto-seeded bracket** — Round of 32 → Final, filled straight from the standings (official FIFA third-place rules).
- **Pick winners** — click a team to advance it; your picks are saved in your browser.
- **Share** — the 🔗 button copies a link that re-creates your bracket for a friend.
- **Save as PNG** — export the full bracket as an image.

## Files
Keep all four together in the same folder:
- `index.html` — the whole app
- `mbappe.avif`, `messi.jpg`, `trophy.svg.webp` — images

## Run / host
- **Local:** double-click `index.html`.
- **GitHub Pages:** upload all four files to a public repo → Settings ▸ Pages ▸ Deploy from `main` / root.

Needs an internet connection for live data and flags. Not affiliated with FIFA or ESPN.
